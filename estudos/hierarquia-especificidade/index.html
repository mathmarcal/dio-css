Hierarquia do Efeito Cascata

Estabelece uma prioridade para a aplicação de uma regra e estilo, caso se tenha mais de uma regra para um mesmo seletor CSS

Se houver um conflito, o navegador irá:

  1. Analisar todas as regras;
  2. Ver o ível de imprtância na origem do elemento;
  3. Comparar a especificidade das declarações com mesmo nível de importância (entra o cálculo de especificidade);
  4. Em caso de empate dp grau de especificidade, irá olhar a ordem de declaração colocada dentor dos arquivos de código. A regra CSS que for declarada por último, irá ser selecionada;

!important

Nível de prioridade do efeito em cascata:

  1. Declarações com !important do usuário;
  2. Declarações com !important do desenvolvedor;
  3. Folha de estilo do desenvolvedor;
    1. CSS inline
    2. CSS interno
    3. CSS externo
    4. Folha de estilos padrões do navegador
