Pseudo-elementos

Usado para adicionar elementos extras na página, sem a necessidada de declara-lo no HTML, ou para selecionar uma pate de algum elemento.

Identificar por ::palavra-chave

Pseudo-elemento - ::first-letter

Seleciona a primeira letra da primeira linha

Propriedades disponíveis:

  • color
  • background
  • margin
  • padding
  • border
  • font
  • Propriedades de texto

Pseudo-elemento - first-line

Seleciona a primeira linha de um texto

Propriedades disponíveis:

  • color
  • background
  • font
  • Propriedades de texto

Pseudo-elemento - ::backdrop

Estiliza plano de fundo de elementos em modo de tela cheia, como dialog ou video

Pseudo-elemento - ::marker

Permite estilizar os tópicos ou numeros de uma lista (li) ou então a flecha de um elemento summary

Propriedades aceitas:
  • color
  • content
  • white-space
  • font
  • animation
  • transition

Pseudo-elemento - ::placeholder

Um placeholder é uma dica para ajudar no preenchimento de algum campo de formulário.

Propriedades aceitas:

  • color
  • font
  • background
  • text

Pseudo-elemento - ::section

Permite estilizar a seleção de texto no navegador

Propriedades aceitas:

  • color
  • background-color
  • text
