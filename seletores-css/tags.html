Seletores CSS

Bem-vindo!

No CSS nós possuimos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor por Classe (.): Busca elementos através do atributo "class"
  • Seletor Universal (*): Seleciona os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo especifíco em nosso documento HTML e conseguimos também buscar atributos com um valor especifíco
